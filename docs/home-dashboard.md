# Dashboard (Bosh sahifa) - Vazifalar va Imkoniyatlar

## Hozirgi Mavjud Funksiyalar

### 1. Ish Vaqti Hisoblagichi
- **Start/Stop/Pause funksiyasi** - ish vaqtini boshlash, to'xtatish va pauza qilish
- **Ish rejimi tanlash** - ofisda yoki masofaviy ishlash variantlari
- **Vaqt hisoblagichi** - real vaqtda ishlash vaqtini ko'rsatish
- **Pauza sababi kiritish** - tushlik, tanaffus va boshqa sabablar uchun
- **Vaqt filtrlash** - kalendar orqali ma'lum muddat bo'yicha filtrlash

### 2. Toast Bildirish Tizimi
- **Success, Error, Warning, Info** toastlar
- **Promise va Loading** toastlar
- **Custom toast** yaratish imkoniyati

### 3. Ishlangan Vaqt Statistikasi
- **Haftalik ish soatlari** diagrammasi
- **Umumiy ishlangan vaqt** ko'rsatkichi
- **Sana bo'yicha filtrlash** imkoniyati

### 4. Vazifalar Ro'yxati
- **Tayinlangan vazifalar** ko'rsatish
- **Vazifa holati** (in_progress, todo) ko'rsatish

## Qo'shilishi Kerak Bo'lgan Funksiyalar

### 1. Ish Vaqti Boshqaruvi
- [ ] **Avtomatik ish kunini boshlash** - tizimga kirganida avtomatik boshlash
- [ ] **Ish vaqti qoidalari** - maksimal ish vaqti, majburiy tanaffus vaqtlari
- [ ] **Ishdan chiqish** vaqtini hisobga olish
- [ ] **Ish jadvalini sozlash** - haftalik ish jadvali belgilash
- [ ] **Offline rejimida ishlash** - internet yo'qligida ham vaqt hisobi

### 2. Kengaytirilgan Statistika
- [ ] **Oylik hisobot** - oylik ish vaqti statistikasi
- [ ] **Produktivlik ko'rsatkichlari** - eng samarali ish vaqtlari
- [ ] **Taqqoslash** - oldingi hafta/oy bilan taqqoslash
- [ ] **Maqsad belgilash** - kunlik/haftalik ish vaqti maqsadlari
- [ ] **Salomatlik ko'rsatkichlari** - ish-tanaffus nisbati

### 3. Vazifalar Boshqaruvi
- [ ] **Yangi vazifa qo'shish** - tezkor vazifa yaratish
- [ ] **Vazifa vaqtini hisobga olish** - har bir vazifa uchun vaqt hisobi
- [ ] **Prioritet belgilash** - muhim vazifalarni ajratish
- [ ] **Deadline notification** - muddat tugash haqida eslatmalar
- [ ] **Vazifa kategoriyalari** - ish turlari bo'yicha kategoriyalash

### 4. Team va Hamkorlik
- [ ] **Team members ko'rsatkichlari** - jamoa a'zolari faolligi
- [ ] **Birgalikdagi vazifalar** - team vazifalarini ko'rsatish
- [ ] **Ish vaqti reyting** - team ichida reyting tizimi
- [ ] **Hamkorlik statistikasi** - team bilan qancha ishlangani

### 5. Qo'shimcha Imkoniyatlar
- [ ] **Harorat va ob-havo** - ish muhitini hisobga olish
- [ ] **Sog'liq ko'rsatkichlari** - ko'z dam olishi, harakat eslatmalari
- [ ] **Musiqa integratsiyasi** - ish paytida musiqa boshqaruvi
- [ ] **AI yordamchisi** - kundalik rejani tavsiya qilish
- [ ] **Kalendar integratsiyasi** - Google Calendar, Outlook bilan ulanish

### 6. Ma'lumotlar va Eksport
- [ ] **PDF hisobot** - kunlik/haftalik hisobotni yuklab olish
- [ ] **Excel eksport** - statistikani Excel formatida eksport qilish
- [ ] **Cloud backup** - ma'lumotlarni bulutga saqlash
- [ ] **Ma'lumotlarni import qilish** - boshqa tizimlardan ma'lumot olib kelish

### 7. Moslashuvchanlik
- [ ] **Custom dashboard** - foydalanuvchi o'z ehtiyojiga moslash
- [ ] **Widget qo'shish** - turli xil vijetlar qo'shish

- [ ] **Dark/Light mode toggle** - mavzu o'zgartirish

### 8. Mobil va Platformalar
- [ ] **PWA support** - mobil ilovaga aylantirish
- [ ] **Desktop notifications** - tizim bildirishlari
- [ ] **Mobile app sync** - mobil ilova bilan sinxronizatsiya
